# `data/synthetic/`

This directory is reserved for clearly-synthetic documents (for example,
hand-written fake "statutes" used to exercise the chunker or the
inference-honesty detector).

It is intentionally empty in this initial release. If you add files
here, put a short note at the top of each one stating that it is
synthetic and not legally binding.

See `docs/data-notes.md` for the full data policy.
